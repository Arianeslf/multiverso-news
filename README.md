# 🌌 Multiverso News

O **Multiverso News** é um portal de notícias focado em astronomia, exploração espacial e curiosidades sobre o universo.
O projeto foi desenvolvido com o objetivo de treinar habilidades em front-end, criando uma interface moderna, interativa e visualmente atrativa.

---

## 🚀 Sobre o projeto

Este site apresenta conteúdos relacionados ao espaço de forma organizada e acessível, com um design inspirado no universo.

Ele foi pensado para:

* 📚 Divulgar informações científicas
* 🌠 Criar uma experiência visual imersiva
* 🧠 Praticar desenvolvimento web (HTML + CSS)

---

## 🎨 Design e Estilo

O layout do site segue uma identidade visual baseada no espaço:

* 🌌 Fundo com gradiente escuro simulando o universo
* ✨ Elementos visuais inspirados em estrelas e galáxias
* 🟣 Cores predominantes: preto, roxo e azul
* 🔤 Tipografia moderna (Poppins e outras)
